Automated Baggage Handling & Tracking System

Role: Lead Systems Integration

In this Computer Engineering project, I architected a Modular C++ codebase to automate luggage sorting and tracking based on real-time data inputs. I developed data parsing algorithms to process passenger information and integrated individual sub-systems into a Master Control Loop to interface with the Quanser Q-Arm API.

Key Technical Contributions

  • System Integration: Managed the software-to-hardware interface, ensuring low-latency communication between sensor data and robotic actuators.
  • Algorithm Design: Engineered a weight-distribution logic that automatically adjusted sorting priority to prevent mechanical strain.
  • Problem Solving: Identified a rotary actuator torque deficiency during testing and proposed a winch-based mechanical redesign.
